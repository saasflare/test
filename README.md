# New Project

Next.js + Tailwind CSS project generated by Craivo

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/` - Next.js App Router pages
- `components/` - Reusable components
- `data/` - Project data and content
- `tailwind.config.ts` - Tailwind configuration with your design tokens
- `licenses/` - IP Assignment and license files
